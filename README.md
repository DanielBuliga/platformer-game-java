# platformer-game-java
Implementation of a simple 2D platformer game developed in Java. The project features basic player movement, including left, right and jump actions, as well as collision detection with platforms and obstacles. A minimal enemy logic system is included to demonstrate interaction between game entities. The game loop follows a standard update–render structure to manage state transitions and drawing operations. Level maps are loaded from external `.txt` files, allowing the environment to be modified without changing the source code. The project illustrates fundamental concepts of object‑oriented programming, modular design and basic game development mechanics.
